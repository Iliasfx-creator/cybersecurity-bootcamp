Concepts
Networks allow systems to exchange data through agreed protocols.
IP addresses identify network endpoints at the logical layer.
MAC addresses identify interfaces on a local network.
Packets contain addressing and protocol information.
Routers move traffic between networks.
Switches connect devices inside a local network.
DNS translates names into network addresses.
Security connection

Security testing requires understanding where traffic originates, where it is
routed, which service receives it, and which trust boundary it crosses.

An address identifies a network destination; it does not prove the identity or
authorization of the user sending the request.

Lesson learned

Networking knowledge is necessary for web testing, proxy configuration,
traffic analysis, segmentation and incident investigation.
