SECURITY CONCLUSION

Property tested:
A discount-related property revealed through the API response schema.

Authorization result:
The client could submit and influence a property that should have been controlled only by the server.

Root cause:
Mass assignment and missing object-property-level authorization.

Remediation:
Calculate discounts server-side, allowlist writable properties, reject unknown properties, and authorize sensitive changes independently.
