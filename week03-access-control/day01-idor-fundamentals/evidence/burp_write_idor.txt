=== Analysis ===

The vulnerable update endpoint authenticated Alice but selected and modified the resource only through the client-controlled document ID.

Alice changed Bob's document because the endpoint omitted a server-side ownership check.

The safe update endpoint repeated the authorization check before modifying the resource and returned 404.

Bob's subsequent safe read confirmed that the rejected update did not alter the document.
