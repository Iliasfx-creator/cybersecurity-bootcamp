QUERY ATTEMPT

CUSTOM HEADER ATTEMPT

BEARER ATTEMPT
GET /me HTTP/1.1
Host: 127.0.0.1:8013
Authorization: Bearer REDACTED

Security conclusion:
Only the cookie transport is implemented. Values in URLs or alternative headers
do not create an authenticated context.
